# execute-battle Delta Specification

## MODIFIED Requirements

### Requirement: CPU同士の戦闘解決
CPU大名同士が戦争を行う場合、ターンごとの戦術選択をスキップし、兵力比に基づく確率的な即時決着を行う。

#### Scenario: CPU間の戦争決着
- **WHEN** 攻撃側・防御側ともにCPUである戦争が発生した時
- **THEN** 両軍の兵力、士気、米の備蓄から算出された勝率に基づき、一括で勝敗が決定される
- **AND** 結果（占領または防衛成功）がアクションログに記録される

### Requirement: 士気高揚（鼓舞）の最適化
CPUは「鼓舞」の戦術を、メリットが明確な場合（士気が著しく低い、またはカウンターとして有効な場合）を除き、原則として選択しない。
