## Why

本プロジェクト（戦国統一）のコアロジックは `engine` に、永続化などのインフラ層は `infrastructure` に分離されていますが、実際に人間がゲームをプレイして通しで動作確認・体験できる単独のフロントエンドアプリケーションがまだ存在していません。ユーザーが一人の大名を選択し、ターミナル上で直感的に内政・戦争プレイを行えるTUI（Text User Interface）クライアントを提供することで、ゲームロジックの検証を容易にし、スタンドアロンなプレイ環境を構築します。

## What Changes

- ワークスペースに新しいクレート `cli` を追加し、`engine` と `infrastructure` に依存させます。
- ターミナルUIライブラリとして `ratatui` と `crossterm` を導入し、描画と入力を管理します。
- **UIステートマシン**: 画面遷移（タイトル、大名選択、内政、戦争）およびサブ状態（数値入力、対象選択、メッセージ表示）を `App` 構造体で管理します。
- 以下の画面とゲームフローを実装します：
  - **タイトル・大名選択画面**: プレイする担当大名を選択。
  - **内政モード（Domestic Mode）**:
    - リソースの管理（金、米、兵、人口、石高、町、忠誠度）。
    - 各種コマンドの実行（米売り、米買い、開墾、町造り、徴兵、解雇、施し）。
    - **輸送**: 隣接国への資源（金、兵、米）の送付。
    - **委任設定**: NPCによる自動内政の切り替え。
    - **他国情報調査**: 情報を得ると共に、ゲーム内時間を進行させる。
  - **戦争モード（War Mode）**: 隣接国への攻撃、戦術（通常、奇襲、火計、鼓舞）の選択。
- **マスターデータ読み込み**: `infrastructure` において、CSVから大名・国・隣接情報をロードする機能を実装し、CLI起動時に使用します。

## Capabilities

### New Capabilities
- `cli-game-client`: プレイヤーが大名を選択し、ターミナル上で内政モードと戦争モードを行き来しながら全国統一を目指す対話型TUIクライアント機能。

### Modified Capabilities
- 該当なし。

## Impact

- **リポジトリ構造**: `Cargo.toml` の `members` に `cli` が追加されます。
- **Engineへの影響**: CLIからユースケースを呼び出す際に、UI表示用に最適化された `UiSnapshot` などのRead Modelを `KuniQueryUseCase` に追加しました。
- **データ管理**: `static/master_data` に `kuni.csv` と `neighbor.csv` が配置され、実行時に読み込まれる必要があります。
- **依存関係**: `ratatui`, `crossterm`, `csv`, `serde` などの関連ライブラリが新たに追加されます。
