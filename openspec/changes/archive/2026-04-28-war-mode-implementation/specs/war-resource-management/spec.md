# Specification: 合戦資源管理 (war-resource-management)

## ADDED Requirements

### Requirement: 出陣時の資源指定
MUST: 攻撃を開始する前に、プレイヤーは自国から持ち出す兵数と米の量を指定できなければならない。

#### Scenario: 兵数と米の入力
- **WHEN** 「戦争」コマンドで攻撃対象国を選択したとき
- **THEN** 投入兵力の入力画面が表示される
- **AND** 兵力確定後、投入米（兵糧）の入力画面が表示される

### Requirement: 資源の減算と保持
SHALL: 合戦に投入された資源は、出撃元の国の在庫から即座に差し引かれ、合戦中の軍勢ステータスとして管理されなければならない。

#### Scenario: 国の資源変動
- **WHEN** 資源入力を確定して合戦を開始したとき
- **THEN** 出撃元の国の「兵」と「米」が入力分だけ減少する
- **AND** 合戦画面にその資源量が「軍勢のステータス」として表示される

### Requirement: 自領攻撃の禁止とフィルタリング
MUST: システムは、自勢力が支配する国への合戦（攻撃）を禁止しなければならない。
SHALL: ターゲット選択リストにおいて、合戦コマンド選択時は他国のみ、輸送コマンド選択時は自国のみが表示されるようにフィルタリングを行わなければならない。

#### Scenario: 自領への攻撃禁止
- **WHEN** 合戦のターゲット選択画面を開いたとき
- **THEN** リストには自分とは異なる大名が支配する隣接国のみが表示される
- **AND** 自分の大名が支配する国は選択肢に含まれない
