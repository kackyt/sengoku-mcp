# Specification: 合戦ループ制御 (battle-loop-control)

## ADDED Requirements

### Requirement: 戦術選択ループ
MUST: 合戦は決着がつくまで、ターンごとに戦術を選択し続けなければならない。

#### Scenario: 継続的な策選択
- **WHEN** 1ターンの戦闘計算が終了し、両軍の兵力が残っているとき
- **THEN** 再び策選択メニューが表示され、プレイヤーは次のターンの策を選べる

### Requirement: 敗走・勝利の判定
MUST: 兵力、食料、または士気のいずれかが0になった時点で合戦は終了し、結果が表示されなければならない。

#### Scenario: 兵糧切れによる敗北
- **WHEN** ターン経過により攻撃軍の「米」が0になったとき
- **THEN** 「兵糧が尽きた！」というメッセージとともに敗北（敗退）が決定する

### Requirement: 退却コマンド
SHALL: プレイヤーは策選択メニューから「退却」を選択し、合戦を途中で切り上げることができなければならない。

#### Scenario: 任意退却
- **WHEN** 策選択で「退却」を選んだとき
- **THEN** 合戦を即座に終了し、残存資源を保持して自国（内政画面）に戻る

### Requirement: 占領後の手番同期
MUST: 合戦に勝利し他国を占領した際、システムはただちにエンジンの手番国とUIの選択国を同期させ、占領した国で即座にコマンドを実行可能にしなければならない。

#### Scenario: 占領直後の操作
- **WHEN** 合戦に勝利して新しい国を支配下に置いたとき
- **THEN** 内政画面に戻った際、占領した国が自動的に選択状態となる
- **AND** その国に対して即座に国内コマンドを実行できる
