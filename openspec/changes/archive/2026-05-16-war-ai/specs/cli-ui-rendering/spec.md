## ADDED Requirements

### Requirement: 合戦ログと侵攻通知のUI描画
CLIは、合戦イベントやCPUからの侵攻をプレイヤーに分かりやすく通知・表示しなければならない (MUST)。

#### Scenario: 合戦ログの描画
- **WHEN** 合戦ターンが進行した時
- **THEN** システムは `ActionLogRenderer` を通じて、ダメージ、士気変動、戦術発動などの合戦ログを、プレイヤーの視点（自軍・敵軍）に合わせて描画しなければならない。

#### Scenario: 侵攻通知モーダルの表示
- **WHEN** プレイヤーへの侵攻が検知された時
- **THEN** CLIは画面中央に視認性の高いモーダルを表示し、どの大名が自国のどの国に攻め込んできたかをプレイヤーに通知しなければならない。
- **AND** プレイヤーが通知を確認した後、自動的に合戦画面（防衛モード）へ遷移しなければならない。
