## ADDED Requirements

### Requirement: 情報コマンドの実行
MUST: プレイヤーは他国の詳細情報を取得することができ、その実行にはコマンド実行権（アクションポイント）を消費しなければならない。

#### Scenario: 他国の情報一覧取得
- **WHEN** プレイヤーが「情報」コマンドを実行したとき
- **THEN** 全ての他国（自分以外の大名）の米、金、兵、石高、町、忠誠度の合計値が一覧で返却される。
- **AND** プレイヤーの今ターンのアクションポイントが1消費される。

#### Scenario: アクションポイント不足時の挙動
- **WHEN** プレイヤーのアクションポイントが0の状態で「情報」コマンドを実行したとき
- **THEN** エラーが返却され、情報は表示されない。アクションポイントは消費されない（あるいは負にならない）。
