## Context

現在のシステムでは、プレイヤーは自国の情報は把握できますが、他国の詳細なステータス（米、金、兵、石高、町、忠誠度）をまとめて取得するインターフェースが不足しています。
戦国シミュレーションゲームとして、他国の戦力を分析し、侵攻先を決定するための「情報」コマンドが必要です。

## Goals / Non-Goals

**Goals:**
- 全ての他国の主要ステータス（米、金、兵、石高、町、忠誠度）を一覧で取得できる機能を実装する。
- コマンド実行により、現在のターンの行動力（コマンド実行権）を消費するようにする。
- MCPツールとして公開し、LLMが戦略的に利用できるようにする。

**Non-Goals:**
- 特定の国だけを詳細に調査する「忍び」のような高度な諜報機能（今回は一括表示のみ）。
- 嘘の情報が混じるなどの不確実性の導入（常に正確な値を表示）。

## Decisions

1. **ユースケースの作成**: `InfoUseCase` を `engine/src/application/usecase` に作成。
    - 他国の情報を集計し、一覧形式のDTOで返却する。
    - 実行時に `Daimyo` の行動力（アクションポイント）を減算する。
2. **行動力消費の統合**: 既存の `DomesticUseCase` 等と同様に、コマンド実行権のチェックと消費をユースケース内で行う。
3. **MCPツールの定義**: `mcp-server` のプレゼンテーション層で `get_other_countries_info` のようなツールを公開する。
4. **情報の範囲**: 現在存在している全ての他国の大名およびその領地の合計値を表示する。

## Risks / Trade-offs

- **ゲームバランス**: 情報取得が常に「1手」を消費するため、そのコストに見合う価値を提供できるかが重要。
- **データ取得コスト**: 全国のデータを走査するため、将来的に国数が膨大になった場合は最適化が必要だが、現時点ではシンプルに全件取得で対応する。
