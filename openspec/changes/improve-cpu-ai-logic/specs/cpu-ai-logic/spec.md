# cpu-ai-logic Specification

## Requirements

### Requirement: 近隣状況に基づく意思決定
CPUは自国のリソースだけでなく、隣接する他国の兵力状況を考慮して行動を決定しなければならない。

#### Scenario: 国防優先の判断
- **WHEN** 隣接する敵国の最大兵力が自軍兵力の1.2倍を超えている時
- **THEN** CPUは「雇用」を最優先で選択する

#### Scenario: 侵攻機会の判断
- **WHEN** 隣接する敵国の最小兵力が自軍兵力の0.7倍以下であり、かつ金・米が十分な時
- **THEN** CPUは「戦争」を選択候補に含める

### Requirement: 優先度ベースのリソース管理
CPUは「兵力 > 金 > 米」の優先順位に従って不足リソースを補う行動をとらなければならない。

#### Scenario: 資金不足時の対応
- **WHEN** 兵力は十分だが、追加の雇用や開発に必要な金が不足している時
- **THEN** CPUは「米売り」または「町造り」を選択する

### Requirement: 戦術カウンターAI
戦闘において、CPUはプレイヤーの過去の行動傾向に基づき、有利な戦術を選択しなければならない。

#### Scenario: プレイヤーの奇襲への対抗
- **WHEN** プレイヤーが過去の戦闘で「奇襲」を多用（50%以上）している時
- **THEN** CPUは「通常（防御）」または「奇襲（防御）」を選択する確率を上げる
