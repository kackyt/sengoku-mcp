# 輸送可能ルート（隣接のみ・1手1区間）

`static/master_data/neighbor.csv` に基づく。兵・金・米は **隣接する自領間のみ** 輸送可能。

## 西日本（安芸集結）

```text
[越州3]──[山城8]──[安芸9]──[四国10]
              │         ├──[豊後11]──[薩摩12]
[甲信4]──[尾張7]─┘         └──（豊後と四国は非隣接）
```

| 送り元 | 安芸へ直送 | 備考 |
|--------|------------|------|
| 山城8 | ✅ | 西軍ハブ |
| 四国10 | ✅ | 豊後へは出撃は安芸 or 豊後から |
| 甲信4 | ❌ → 8経由 | 2手 |
| 尾張7 | ❌ → 8経由 | 2手 |
| 越州3 | ❌ → 8経由 | 2手 |
| 蝦夷1・奥州2・武蔵5・三河6 | ❌ | 8/9と非隣接。多段が必要 |

## 東北（奥州・蝦夷）

```text
[蝦夷1]──[奥州2]──[越州3]
              └──[武蔵5]
```

奥州→安芸: 最低 `2→3→8→9` で **3ターン**（輸送のみ）。

## 手番待ちの安全コマンド

auto を使わないとき、**兵0・敵隣接なし**の国:

- `domestic_rice_sell`（amount 1 など最小）
- 施し（忠誠 &lt; 50 のときのみ）

## 集結先の選び方

| 目的 | 推奨集結先 |
|------|------------|
| 安芸・豊後方面 | **安芸9** |
| 徴募してから送る | **山城8**（人口・金が多いことが多い）→ 安芸 |
| 東北の兵 | 奥州に一旦集めず、**奥州50維持**しつつ蝦夷→奥州→越州→山城 |
