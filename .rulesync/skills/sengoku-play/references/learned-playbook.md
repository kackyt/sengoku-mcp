# プレイ実績からの学び（全国統一まで）

織田・武力重視プレイで全国統一に至るまでの失敗と回復をまとめた。**手動プレイ・スプリント運用時に必ず参照**。

## 最重要ルール（違反＝裏切り）

| ルール | 内容 |
|--------|------|
| **主君との合意はご法度** | 兵力・米・宣戦・戦術・Done条件を勝手に変えない。「足りそう」で先走らない |
| **GOなき宣戦禁止** | ゲート未達でも `battle_start_war` しない |
| **準備中 `domestic_auto_action` 禁止** | 自領の auto が **CPU侵攻**を誘発（例: 安芸→四国）。手番待ちは `domestic_rice_sell` 等の安全コマンド |
| **1国1手・その国の手番のみ** | 他国IDで輸送・宣戦すると失敗。連打しない |
| **施し** | **忠誠 &lt; 50 のときだけ**（50超は主君が明示しない限り不要） |

## 宣戦前ゲート（毎回、表で提示してからGO）

```text
□ 攻撃元の 兵 ___ / 目標 ___
□ 米 ___ / 出陣予定 ___
□ 戦術: 通常（必勝戦は初手奇襲・鼓舞なし）
□ 主君GO: 済 / 未
□ auto_action: 未使用
```

**集結スプリント**では「宣戦」より先に **数値ゲート達成** を Done にする（例: 安芸兵400・米200）。

## 合戦戦術

| 状況 | 初手 | 理由 |
|------|------|------|
| **必勝が必要**（奪還・制圧・全国統一） | **通常(1)** | 奇襲(2)は読まれて一撃全滅の実績あり（兵200 vs 敵152でも敗北） |
| 余裕があり試す | 主君GO後に奇襲 | 「かっこよさ」より Done |
| 士気回復が目的 | 鼓舞(4) | 畳み切り用ではない |
| 防衛・劣勢 | 通常 or 奇襲 | 状況を見て。空城防衛は即敗 |

## 輸送・集結

- **隣接国のみ** 1手で輸送。非隣接は **多段**（1国1ターン）。
- 東北→安芸: `蝦夷→奥州→越州→山城→安芸` または `奥州→越州→…`
- **四国は安芸としか隣接しない**（豊後とは非隣接）。西軍の合流は **四国→安芸**。
- **山城・安芸は非隣接でない** — 山城→安芸は直送可。山城と四国は非隣接。
- 詳細: [transport-map.md](./transport-map.md)

### 集結の典型（西進）

| 段階 | 内容 |
|------|------|
| 1 | 四国: 施し（忠誠50+）→ 兵を安芸へ全送り |
| 2 | 山城: 徴募 → 兵を安芸へ全送り（または安芸で徴募） |
| 3 | 東方: 遠い国から兵を段階輸送 |
| 4 | 諜報: **兵0・敵隣接なし**の国の手番で `get_other_countries_info` |
| 5 | ゲート達成後、主君GOで宣戦 |

## 守備・占領

| 項目 | 基準 |
|------|------|
| 新規占領直後 | **守備50+** を確保（徴募 or 勝ち残り） |
| 空城 | 敵隣接なら **即奪還** の実績（山城兵15→武田） |
| 「ほぼ全送り」 | 主君が明示した **内陸・味方囲み** のみ。前線空城はNG |
| 奥州 | 主君方針で **兵50前後を死守** しがち — 全送りしない |

## スプリント実行の型

戦国プレイは **必ずスプリント運用**。期間は **標準4ターン（1年）**。詳細手順は [sprint-cadence.md](./sprint-cadence.md)。

1. **主君と Done を一行で固定**（例: 「安芸奪還」「兵400集結」）
2. **期間**（標準4ターン）と **数値ゲート**（兵・米・忠誠）を表にする
3. **準備フェーズ** — 宣戦なし、auto禁止
4. **攻撃フェーズ** — 別途GO（「○○へ出撃GO」）
5. **クローズ** — Done達成を報告 → レビュー → レトロ。勝手にスコープ変更しない

> マイルストーン（M1〜M4 等の中期ロードマップ）は **主君要望時のみ** 導入する任意概念。小田中から押し付けない。詳細は [sprint-cadence.md のマイルストーン節](./sprint-cadence.md#マイルストーン任意主君要望時のみ)。

### 実績に基づく敵規模（参考・必ず諜報）

| 時期 | 豊後 | 薩摩 |

## よくある失敗パターン

| 失敗 | 結果 | 対策 |
|------|------|------|
| 集結前に宣戦 | KPI未達のまま戦争 | ゲート表を先に |
| 奇襲初手（必勝戦） | 一撃全滅 | 通常で削る |
| auto_action（安芸等） | 合意外侵攻・連鎖 | 準備中禁止 |
| 手番違いの輸送連打 | 何も進まない | 1国1手 |
| 「勝手に自領」誤認 | 報告の誤り | 侵攻ログ・諜報で確認 |

## 振り返りフォーマット

主君が好む形式:

- **FUN / DONE / LEARN** — スプリント総括
- **薔薇 / 棘 / 蕾** — 短い感情振り返り（KPTは使わない）
