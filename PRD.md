# 戦国統一 - 外部仕様書

## ゲーム概要
「戦国統一」は、MCPサーバーとして開発された戦国時代を舞台とする戦略シミュレーションゲームです。プレイヤーは大名となり、領地の管理、軍事行動、外交などを通じて全国統一を目指します。

## ゲーム状態

ゲームは主に以下の2つの状態（モード）を持ちます：

1. **戦争モード** - 他の大名と戦闘を行うモード
   - プレイヤーは兵力、資金、食料を消費して他国を攻撃
   - 戦闘では「通常攻撃」「奇襲」「謀略」「撤退」などの戦術を選択可能
   - 勝利すると領土を獲得

2. **内政モード** - 領地の管理や開発を行うモード
   - 米売り、米買い、開墾などのコマンドで国力を強化
   - 資源（金、米、兵）のバランス管理が重要

### モード遷移

- ゲーム開始時は内政モードから始まる
- 内政モード中に「戦争」コマンドを選択すると戦争モードに移行
- 戦争の結果表示後、自動的に内政モードに戻る
- 各大名は順番に行動し、全大名の行動が終わると季節（ターン）が進む

## UI要素

### 内政モード UI
1. **ステータス表示** - 画面上部に表示される以下の情報
   - 金（資金）
   - 兵（兵力）
   - 米（食料）
   - 人口
   - 石高（領地の価値）
   - 町（都市数）
   - 忠誠度
2. **コマンドメニュー** - 画面下部に表示される以下のコマンド
   - **戦争** - 戦争モードに移行し、隣接する敵国を選択して攻撃
   - **米売り** - 米を消費して金を得る
   - **米買い** - 金を消費して米を得る
   - **開墾** - 金を消費して石高（領地価値）を上げる
   - **町造り** - 金を消費して町（都市）を増やす
   - **雇用** - 金を消費して兵を増やす（人口が減少）
   - **解雇** - 兵を減らして忠誠度と人口を増やす
   - **施し** - 米を消費して忠誠度を上げる
   - **輸送** - 他の自国領地へ資源を送る
   - **委任** - 他の自国領地をCPUに委任する
   - **解任** - 委任した領地の委任を解除する
   - **情報** - 他国の情報を調査して表示する
   - **終了** - ゲームを終了する
3. **メッセージ表示** - 画面中央に表示される国の状況や季節情報

### 戦争モード UI
1. **軍勢情報表示** - 両軍の以下の情報
   - 兵力
   - 食料
   - 士気
2. **戦術コマンド** - 戦闘時に選択できる以下の戦術
   - **通常** - 標準的な攻撃（防御時は「堅守」）
   - **奇襲** - 敵の士気を下げる攻撃
   - **火計** - 敵の食料を減らす作戦
   - **鼓舞** - 士気を上げる戦術
3. **戦況表示** - 戦闘の進行状況を表示

## 操作方法と各コマンドの振る舞い

### 基本操作
- **方向キー** - メニュー選択、カーソル移動
- **決定ボタン** - コマンド実行、選択確定
- **ソフトキー** - 状況に応じた操作（戻る、決定など）
- **数字キー** - 数値入力（資源量や攻撃対象の選択）

### 内政モードのコマンド振る舞い
- **戦争** - 隣接する敵国を数字で選択し、投入する兵力・資金・食料を順に指定
- **米売り** - 消費する米の量を指定すると、ランダムな倍率で金を獲得
- **米買い** - 消費する金の量を指定すると、米を獲得
- **開墾** - 消費する金の量を指定すると、石高（領地価値）が上昇
- **町造り** - 消費する金の量を指定すると、町（都市数）が増加
- **雇用** - 消費する金と人口を使って兵を増加させる（忠誠度が減少）
- **解雇** - 減らす兵の量を指定すると、忠誠度と人口が増加
- **施し** - 消費する米の量を指定すると、忠誠度が上昇
- **輸送** - 他の自国領地を選択し、送る資源（兵・金・米）の量を指定
- **委任** - 委任する自国領地を選択すると、その領地はCPUが管理
- **解任** - 解任する委任領地を選択すると、プレイヤーの管理に戻る
- **終了** - ゲームを終了してタイトル画面に戻る

### 戦争モードのコマンド振る舞い
- **通常** - 標準的な攻撃/防御を行う（相手の戦術により効果が変動）
- **奇襲** - 敵の士気を下げる攻撃（通常防御に対して効果的）
- **火計** - 敵の食料を減らす作戦（火計防御以外に効果的）
- **鼓舞** - 自軍の士気を上げる防御戦術

### 戦術の組み合わせ効果

各戦術の組み合わせによる詳細な効果：

#### 攻撃側：通常
- vs 通常防御：通常ダメージ（倍率180、ハードモード時100）
- vs その他防御：低ダメージ（倍率60、ハードモード時40）

#### 攻撃側：奇襲
- vs 通常防御：高効果（倍率40、ハードモード時20）+ 敵士気低下、自軍士気上昇
- vs 奇襲防御：低効果（倍率300、ハードモード時180）+ 自軍士気低下
- vs その他防御：高効果（倍率30、ハードモード時20）+ 敵士気低下、自軍士気上昇

#### 攻撃側：火計
- vs 火計防御：自軍兵力減少（20-40%、ハードモード時20-40%）+ 自軍士気低下
- vs その他防御：敵食料大幅減少（30-80%、ハードモード時40-80%）+ 敵士気低下、自軍士気上昇

#### 防御側：鼓舞
- 自軍士気上昇（5-20ポイント）

## ゲーム進行

### ターン制システム
- ゲームは各大名が順番に行動するターン制で進行します
- 各ターンの開始時に、大名の行動順序がランダムに決定されます
- すべての大名が行動を終えると、1ターン（季節）が経過します
- ターンカウンター（`turn`変数）が増加し、次のターンが始まります

### プレイヤーとCPUの行動
- プレイヤーが操作する大名の番になると、コマンドメニューから行動を選択できます
- CPU大名の番になると、自動的に以下のような行動を取ります：
  - 資金が十分にある場合：米売り、米買い、雇用などのコマンドを実行
  - 兵力が十分にある場合：隣接する敵国に攻撃を仕掛ける
  - 資源が不足している場合：資源を回復するコマンドを優先的に実行
- CPU大名の行動は、その大名の資源状況や周囲の状況に応じて決定されます
- CPU大名の行動は画面上に表示され、プレイヤーはその結果を確認できます

### 季節イベント
ゲーム内では4ターンごとに以下の季節イベントが発生します：

1. **人口増加**（`turn%4==0`の時）
   - 各国の人口が10〜12%増加します
   - 計算式：`jinko += jinko * (random(3)+10) / 100`

2. **資源生成**（`turn%4==2`の時）
   - 各国の金と米が自動的に増加します
   - 金の増加：忠誠度、人口、町の数に比例
   - 米の増加：忠誠度、人口、石高に比例
   - 計算式：
     - `kin += tyu*(random(2)+3)/100 + jinko*(random(5)+10)/100 + machi*(random(15)+25)/100`
     - `kome += tyu*(random(2)+3)/100 + jinko*(random(5)+10)/100 + kokudaka*(random(15)+25)/100`

3. **緊急事態**（ターン開始時にランダムで発生）
   - **疫病**（`turn%4==1`の時）：影響を受けた国の人口が5〜9%減少
     - 計算式：`人口 -= 人口 * (random(5) + 5) / 100`
     - 他の被害：食料10〜29%減少、忠誠度20〜29%減少、町20〜39%減少、石高20〜39%減少
   - **飢饉**（その他の季節）：影響を受けた国の人口が5〜19%減少
     - 計算式：`人口 -= 人口 * (random(15) + 5) / 100`
     - 他の被害：兵力15〜34%減少、忠誠度15〜24%減少、町5〜14%減少、石高5〜14%減少
   - 発生確率：各国ごとに1/40の確率で発生
   - 複数の国が同時に被害を受ける可能性があります

これらの季節イベントは国力のバランスに大きく影響するため、プレイヤーは長期的な戦略を立てる必要があります。

## 資源計算式

各コマンドの詳細な計算式：

### 米売り
- 消費：`米 -= 投入量 * BIAS`
- 獲得：`金 += 投入量 * (random(BIAS) + BIAS)` (1.0〜2.0倍のランダム倍率)

### 米買い
- 消費：`金 -= 投入量 * (random(BIAS) + BIAS)` (1.0〜2.0倍のランダム価格)
- 獲得：`米 += 投入量 * BIAS`

### 開墾
- 消費：`金 -= 投入量 * BIAS`
- 獲得：`石高 += 投入量 * (45 + random(10))` (45〜54倍)

### 町造り
- 消費：`金 -= 投入量 * BIAS`
- 獲得：`町 += 投入量 * (45 + random(10))` (45〜54倍)

### 雇用
- 消費：`金 -= 投入量 * BIAS/2`, `人口 -= 投入量 * BIAS`, `忠誠度 -= 投入量 * BIAS/2`
- 獲得：`兵 += 投入量 * BIAS`

### 解雇
- 消費：`兵 -= 投入量 * BIAS`
- 獲得：`忠誠度 += 投入量 * BIAS/2`, `人口 += 投入量 * BIAS`

### 施し
- 消費：`米 -= 投入量 * BIAS`
- 獲得：`忠誠度 += 投入量 * (BIAS/2 + random(BIAS/2))` (0.5〜1.0倍のランダム倍率)

### 戦闘勝利条件

戦闘は以下の条件で決着が付きます：

- **敗北条件**：兵力、食料、士気のいずれかが0以下になった時
- **勝利時**：敵国の領土を獲得し、残存兵力と食料を統合
- **敗北時**：投入した兵力と食料を失い、残存分のみ回収
- **毎ターン消費**：兵力の30%の食料を消費、士気が自然減少
- **食料不足時**：士気が大幅に低下（30-50ポイント減少）
